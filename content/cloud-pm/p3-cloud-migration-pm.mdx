## מה תלמד

בסוף המודול הזה אתה מסוגל:
- לבנות Migration Playbook עם AI ל-Cloud Migration Project
- לבצע TCO Analysis מעמיק שכולל עלויות נסתרות
- לכתוב Cutover Plan ל-Migration Weekend

---

## מקורות לימוד

> **Cloud Migration ב-2025:** ממוצע מהגירה ל-Enterprise לוקח 18-24 חודשים. AI יכול לקצר את שלב התכנון ולהפחית Surprises.

**על Cloud Migration:**
- [AWS Migration Hub](https://aws.amazon.com/migration-hub/) - כלי מהגירה ל-AWS
- [Azure Migrate](https://azure.microsoft.com/en-us/products/azure-migrate/) - הערכה ומהגירה ל-Azure
- חפש: `"cloud migration project management AI 2025"` - AI ב-Cloud Migration PM
- חפש: `"TCO cloud migration hidden costs 2025"` - עלויות נסתרות במהגירה

**על Cutover:**
- חפש: `"cloud migration cutover plan template 2025"` - תבניות Cutover Plan
- חפש: `"cloud migration rollback strategy 2025"` - אסטרטגיות Rollback

## מטלת הגשה

**Migration Package. עמוד וחצי.**

1. **Playbook Highlights:** הצג את 3 הPhases הראשיים עם Milestones ו-Go/No-Go לכל אחת
2. **TCO Surprises:** מה ה-AI מצא ב-TCO שלא חשבת עליו?
3. **Cutover Plan:** הצג Cutover Timeline (6 שעות) עם Checkpoints
4. **Rollback:** מתי תפעיל Rollback? מה ה-trigger?
