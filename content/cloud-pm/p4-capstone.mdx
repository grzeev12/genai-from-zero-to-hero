## מה תלמד

Capstone זה Specialization בניהול Programs ו-PMO. אתה יוצא עם Package שניתן להציג ל-CTO.

---

## מה נדרש

**שלושה תוצרים:**

**1. Program Plan (40%)**
- Program שכולל 3 פרויקטים קשורים
- Program Roadmap עם Dependencies
- Risk Register ב-Program Level
- Governance Structure

**2. AI PMO Blueprint (35%)**
- כיצד PMO שלך ישתמש ב-AI
- Tools, Workflows, Automation
- KPIs ו-Dashboard

**3. PM Guide (25%)**
- Guide לPM שמתחיל ב-AI: מה לעשות, מה לא
- Best Practices מנסיונך
- Lessons Learned

## כלים לחקור

- **Claude** - "עזור לי לכתוב Program Charter ל-Program שכולל [3 פרויקטים קשורים]. כלול: Program Objectives, Success Criteria, Governance, Dependencies, Program Risks."
- **Claude** - "כתוב PM AI Guide ל-Junior PM שמתחיל בעבודה. 1-2 עמודים. כלול: מה AI עוזר בו, מה הגבולות, Best Practices, Common Mistakes."

## מטלת הגשה

**Program Package. 5-7 עמודים.**

**Program Plan (2-3 עמודים):**
- Program Charter (עמוד)
- Program Roadmap - Timeline עם 3 פרויקטים
- Dependencies Map
- Program Risks (5+ Program-Level Risks)

**AI PMO Blueprint (עמוד וחצי):**
- כיצד AI משולב בכל תהליך PMO
- Automation Opportunities
- Dashboard Design

**PM Guide (עמוד):**
- Guide לJunior PM
- What AI Can / Cannot Do
- Golden Rules
