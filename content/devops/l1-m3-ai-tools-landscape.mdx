## המסגרת

לכל כלי AI יש חוזקות שונות. מהנדס דבופס שבוחר כלי אחד בלבד לכל המשימות מפספס.

**ChatGPT**: ממשק עם GPTs מותאמים, Plugins, ו-Code Interpreter, שמריץ קוד בסביבה מבודדת. שימושי לבדיקת סקריפט לפני הרצה אמיתית.

**Claude**: חזק בניתוח קבצי קונפיגורציה ארוכים (Terraform modules שלמים, Helm charts מרובי קבצים). Projects שומרים context קבוע. Artifacts מציגים קוד בפאנל נפרד לעריכה נוחה.

**Microsoft Copilot**: אינטגרציה טבעית עם כלי Microsoft (Azure DevOps, VS Code, Teams). שימושי בארגונים שכבר על אקוסיסטם Microsoft.

**מתי לבחור מה:**
- ניתוח קובץ ענק (מעל 5,000 שורות): Claude
- בדיקת קוד עם הרצה בפועל: ChatGPT + Code Interpreter
- זרימת עבודה משולבת ב-Azure: Copilot

**כלל הזהב לפרטיות:** Secrets, API Keys, סיסמאות, ומידע פנימי רגיש, לעולם לא לכלי AI ציבורי בגרסת Free. Enterprise tier של כל הכלים מבטיח (בדרך כלל) zero data retention, אבל תמיד לוודא בהגדרות.
