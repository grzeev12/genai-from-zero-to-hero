## המסגרת

Copilot, Claude Code, Codex ו-Cursor עובדים ישירות ב-IDE ובטרמינל, לא רק בצ'אט חיצוני. זה משנה את קצב העבודה.

**התקנה והגדרה:** כל הכלים דורשים auth דרך הארגון או חשבון אישי. ב-VS Code/IDE: extension + login (Copilot, Cursor). בטרמינל: Claude Code ו-Codex מספקים chat ישיר עם context מלא של הפרויקט, כולל הרשאה להריץ פקודות ולערוך קבצים בעצמם.

**Inline Suggestions:** הצעות אוטומטיות תוך כדי כתיבה. שימושי לפונקציות שגרתיות, אבל תמיד לבדוק שההצעה מתאימה להקשר הספציפי, לא רק "עובד".

**Chat מובנה / סוכן טרמינל:** שאלות מורכבות יותר ("איך לכתוב טסט לפונקציה הזאת?", "תקן את הבאג הזה בכל הפרויקט") עדיפות ב-Chat או בסוכן טרמינל (Claude Code/Codex), לא ב-Inline.

**כתיבת טסטים אוטומטיים:** Terratest ל-Terraform, PyTest ל-Python. AI מצוין ביצירת structure ראשוני, אתה מוודא כיסוי של מקרי קצה.

**Refactoring קוד CI/CD קיים:** סוכני הפיתוח מצוינים בזיהוי קוד מיושן או לא יעיל בפייפליינים קיימים, ויכולים לבצע שינויים רב-קבציים בעצמם.

**סיכון רישוי:** קוד שנוצר על ידי AI עלול "לזכור" קטעים מפרויקטי open-source עם רישיון מגביל. ל-repos ארגוניים, בדוק מדיניות רישוי.
