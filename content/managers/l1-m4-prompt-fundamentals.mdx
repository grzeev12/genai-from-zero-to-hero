## המסגרת

הבדל בין שאלה ל-prompt: שאלה מבקשת מידע. Prompt מגדיר תפקיד, הקשר, משימה ופורמט.

**מסגרת RCTF:**

**R - Role (תפקיד)**
"אתה [תפקיד] עם [ניסיון] שעובד ב[הקשר]."
- חלש: "עזור לי לכתוב מייל"
- חזק: "אתה מנהל תקשורת ב-enterprise עם 15 שנה ניסיון בניסוח מסרים לרמת C-suite"

**C - Context (הקשר)**
המידע שה-AI לא יכול לדעת בלעדיך:
- מי אתה ומה תפקידך
- מה הסיטואציה הספציפית
- מה קרה לפני
- מי הקהל

**T - Task (משימה)**
פועל + תוצאה ברורה. לא "עזור לי עם" - אלא "כתוב / נתח / השווה / תכנן".
- חלש: "תעזור לי עם הספק"
- חזק: "נתח את ההצעה המצורפת וזהה 3 סעיפים בעייתיים שיש לדון בהם לפני חתימה"

**F - Format (פורמט)**
כיצד אתה רוצה את הפלט:
- bullet points / טבלה / טיוטת מייל / executive summary / שאלות לפגישה
- אורך: "לא יותר מ-200 מילים" / "5 bullet points"
- שפה: עברית / אנגלית / מעורב

**Constraint (אופציונלי)**
מה לא לכלול, מה להימנע ממנו, מה ייחודי לסיטואציה.

**תבנית מוכנה:**
> "אתה [R]. [C - תאר את הסיטואציה]. [T - פועל + תוצאה]. הצג [F - פורמט]."
